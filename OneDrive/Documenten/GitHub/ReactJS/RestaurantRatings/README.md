# Restaurant Ratings - Рейтинг ресторанов

Приложение для просмотра и оценки ресторанов.

## Структура проекта

```
src/
  ├── api/              # API запросы
  │   └── restaurants.ts
  ├── components/
  │   ├── ui/          # UI компоненты
  │   │   ├── Avatar/
  │   │   ├── Logo/
  │   │   ├── RestaurantCard/
  │   │   └── Search/
  │   └── business/    # Бизнес-компоненты
  │       └── RestaurantList/
  ├── hooks/           # Кастомные хуки
  │   ├── useRestaurants.ts
  │   └── useUpdateRating.ts
  ├── pages/           # Страницы
  │   └── HomePage/
  ├── types/           # TypeScript типы
  │   └── index.ts
  ├── App.tsx
  └── main.tsx
```

## Установка и запуск

```bash
npm install
npm run dev
```

## Особенности реализации

- **Чёткая структура**: разделение на UI-компоненты, бизнес-компоненты, хуки и API
- **Управление состоянием**: все запросы к API выполняются из компонента страницы
- **Контролируемые компоненты**: поиск реализован как контролируемый компонент
- **Обработка состояний**: загрузка, ошибки и пустые состояния обрабатываются корректно
